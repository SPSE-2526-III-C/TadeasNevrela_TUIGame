# SECURITY TERMINAL: RAYCAST HORROR (C)

A terminal-based security operator simulation horror game built in C.

The player does not move through a world directly. Instead, they operate a facility through a structured security terminal interface featuring menu-driven systems and pseudo-3D raycasted camera feeds rendered in ASCII.

---

# 🧠 CORE CONCEPT

You are a night security operator monitoring a facility through a terminal interface.

Everything is controlled via:
- Cursor-based menus
- System panels (cameras, doors, logs, reports)
- Raycasted surveillance cameras
- System integrity monitoring

The horror comes from:
- system corruption
- unreliable information
- contradictory reality
- loss of control

The system itself gradually becomes the antagonist.

---

# 🖥️ UI SYSTEM

The interface behaves like a real security operating system.

Everything is menu-driven and controlled through a cursor.
